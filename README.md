### 1. Clone o repositório e entre na pasta

```bash
git clone https://github.com/uictor3/devopsPets.git
cd devopsPets
cd cadastroPet-back
cd infra
```

### 2. Suba a aplicação com Docker Compose

```bash
docker-compose up --build
```

Esse comando irá:

- Fazer o build e executar o docker

### 3. Acesse o site

```
http://localhost:3000/
```
